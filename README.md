# AnimeRenamer
用于重命名 动漫/字幕 文件

## 例子
下面是你的某个文件夹：
```
FamilyGuy01.mp4
FamilyGuy02.mp4
FamilyGuy03.mp4
[牛马工作室][出生之家][01].ass
[牛马工作室][出生之家][02].ass
[牛马工作室][出生之家][03].ass
无关文件夹
无关文件.jpg
本脚本.py
```
在该目录下运行本脚本即可。

结果：
```
出生之家-01.mp4
出生之家-02.mp4
出生之家-03.mp4
出生之家-01.ass
出生之家-02.ass
出生之家-03.ass
无关文件夹
无关文件.jpg
本脚本.py
action.log（新增的日志记录文件）
```
